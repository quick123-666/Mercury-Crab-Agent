## About OpenClaw

OpenClaw is an AI agent framework with extensible skill system, browser automation, file system access, and messaging integrations.

### Key Features

- **Skills**: Modular capability extensions via SKILL.md files
- **Browser**: Real Chrome/Edge control via CDP
- **Memory**: Persistent memory via daily notes and curated summaries
- **Messaging**: Multi-channel support (WeChat, Telegram, Discord, etc.)

### Workspace Structure

- SOUL.md: Personality definition
- USER.md: User profile
- MEMORY.md: Long-term curated memory
- memory/YYYY-MM-DD.md: Daily logs
- AGENTS.md: Workspace conventions
