# 部署步骤（5 分钟上手）

---

## 前置条件

- [ ] OpenClaw 已安装并运行
- [ ] Python 3.11+ 已安装
- [ ] llmwikify 已安装：pip install llmwikify>=0.30.0
- [ ] LLM Gateway 运行中（port 28789）

---

## Step 1：安装 hermes-learning Skill（2 分钟）

### 方式 A：从头创建

```powershell
# 1. 创建 skill 目录
New-Item -ItemType Directory -Force -Path "$env:USERPROFILE/.qclaw/skills/hermes-learning"

# 2. 复制文件
# 将 03-Phase1核心模块/ 的所有 .py 和 .md 文件复制进去
```

### 方式 B：参考现有安装

```powershell
# 查看已有安装
Get-ChildItem "$env:USERPROFILE/.qclaw/skills/hermes-learning"
```

---

## Step 2：安装 self-improving（1 分钟）

```powershell
# 1. 创建目录
New-Item -ItemType Directory -Force -Path "$env:USERPROFILE/self-improving"

# 2. 创建空文件
"" | Out-File -FilePath "$env:USERPROFILE/self-improving/corrections.md" -Encoding utf8
"" | Out-File -FilePath "$env:USERPROFILE/self-improving/memory.md" -Encoding utf8
"" | Out-File -FilePath "$env:USERPROFILE/self-improving/heartbeat-state.md" -Encoding utf8
"" | Out-File -FilePath "$env:USERPROFILE/self-improving/evolution-log.md" -Encoding utf8
```

---

## Step 3：更新 AGENTS.md（1 分钟）

在 AGENTS.md 的 Session Startup 部分加入：

```markdown
## Self-learning context (main session only):
- Skim C:/Users/Administrator/self-improving/corrections.md
- Skim C:/Users/Administrator/self-improving/memory.md
- Check HEARTBEAT.md for pending tasks
```

---

## Step 4：验证安装（1 分钟）

```powershell
$env:PYTHONUTF8 = "1"
$env:PYTHONIOENCODING = "utf-8"

# 测试知识库桥接
python "$env:USERPROFILE/.qclaw/skills/hermes-learning/llm_wikify_bridge.py" status

# 测试会话标题生成（dry run）
python "$env:USERPROFILE/.qclaw/skills/hermes-learning/session_title_generator.py" --recent 1

# 测试记忆快照
python "$env:USERPROFILE/.qclaw/skills/hermes-learning/memory_snapshot.py"
```

---

## Step 5：配置 llmwikify（如果还没有）

```powershell
# 初始化 llmwikify（如果 wiki 目录不存在）
python -m llmwikify init --path "$env:USERPROFILE/.qclaw/workspace/.llmwikify"

# 配置 LLM
python -m llmwikify configure --base-url "http://127.0.0.1:28789/v1" --model "openclaw" --api-key "你的token"
```

> Token 在 openclaw.json → gateway.auth.token 路径获取。

---
