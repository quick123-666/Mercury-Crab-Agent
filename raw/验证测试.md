## 常见失败处理

### 症状：知识库命令报 OSError 10106

**原因**：pydantic_core 在 Python 3.13 上有兼容性问题

**解决**：用 subprocess 方式调用（已在 llm_wikify_bridge.py 中处理）

### 症状：中文输出乱码

**原因**：没有设置 UTF-8 环境变量

**解决**：
```powershell
$env:PYTHONUTF8 = "1"
$env:PYTHONIOENCODING = "utf-8"
```

### 症状：LCM DB 查询失败

**原因**：DB 路径不对或 DB 不存在

**解决**：确认 ~/.qclaw/memory/lossless/lcm.db 存在

### 症状：LLM 调用超时

**原因**：Gateway 响应慢（正常 5-8s）

**解决**：脚本内 timeout 已设为 60s，等待即可
