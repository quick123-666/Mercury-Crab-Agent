## LLM 服务配置

本系统依赖本地 LLM 服务（Gateway 模式）：

| 配置项 | 值 | 说明 |
|--------|-----|------|
| 协议 | HTTP | |
| 地址 | http://127.0.0.1:28789/v1 | 本地 Gateway，注意 /v1 后缀 |
| 模型名 | openclaw | 不是 modelroute，不是 19000 |
| 认证 | Bearer Token | 从 openclaw.json → gateway.auth.token 读取 |

> ⚠️ 常见错误：填了端口 19000 或模型名 modelroute，导致 LLM 调用失败。
> 正确值只有 127.0.0.1:28789 + model=openclaw。

## Windows 编码环境变量（必须）

所有 Python 脚本在 Windows 上运行前，需设置编码环境变量：

```powershell
$env:PYTHONUTF8 = "1"
$env:PYTHONIOENCODING = "utf-8"
```

或在脚本内开头加：
```python
import sys, io
sys.stdout = io.TextIOWrapper(sys.stdout.buffer, encoding="utf-8")
```
